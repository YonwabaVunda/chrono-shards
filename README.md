  - Esc key works as pause and resume.

-  How to setup the Game
   -  git clone https://github.com/YonwabaVunda/chrono-shards.git
   -  cd chrono-shards
   -  npm install --save three
   -  npm install --save-dev vite
   -  npx vite or npm run dev (to run the Game)
